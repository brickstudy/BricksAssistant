# **Brickstudy 소개**


안녕하세요. Brickstudy입니다.

저희 Brickstudy는 데이터/AI 프로젝트 모임으로 데이터 직군으로 성장하려는 사람들이 모여 함께 고민하고 성장하는 것을 목표로하는 모임입니다.


## **Motivation**

- 데이터 직군(DE, DS, AI, 서비스 기획 등)의 경우 다른 일반 개발 직군에 비해 혼자서 프로젝트를 진행하는데 많은 시간, 비용적인 부분에 어려움이 있습니다.
- 이러한 문제를 해결하고자 공통된 데이터 플랫폼을 기반으로 여러 프로젝트를 진행 및 협력하여 더 높은 수준의 프로젝트를 진행할 수 있는 모임을 진행하고자 **Brickstudy**라는 모임을 기획했습니다.


## **Meaning of the Name "Brickstudy”**

- Brickstudy는 벽돌을 의미하는 `Brick`과 공부를 의미하는 `Study` 를 합쳐서 만들어진 이름입니다.
- ***Brick***
    - 단단하고 하나하나 쌓아 올려야 하는 벽돌처럼, 각 회원들이 자신들의 프로젝트와 노력을 하나씩 쌓아가며 탄탄한 기반을 만들고자 하는 의미를 담고 있습니다.
- ***Study***
    - 지속적인 학습과 성장을 추구하는 모임의 목표를 반영합니다.


## **Key Activities**

- 개별 프로젝트(AI, 스터디, 연구 등) 진행 및 리딩
- 매주 온라인 공유회를 통한 지식 공유
- 월 1회 오프라인 공유회를 통한 프로젝트 발표
- 데이터 엔지니어와 데이터 플랫폼을 활용한 고수준의 프로젝트 진행


## **Infra**

- 공용 PC : Airflow, Kafka
- 클라우드 : AWS S3, RDS(MySQL), DocumentDB, Lambda, MLFlow
- 협업 : Github, Discord(+Brickassitant), Tistory(blog)


## **Member Roles**

### **Bricks member**
- Brickstudy 모든 인프라 사용 권한 및 활동에 참여하는 인원
- 세부 역할
    - Project Leader : Brickstudy 인프라를 활용하여 프로젝트 개설할 권리를 가진 인원
    - Data Engineer : Brickstudy 인프라를 관리하며, ML/Data 파이프라인을 관리하는 인원
- 주요 활용
    - 프로젝트 진행(ex. ML/데이터 인프라 사용, Project member 선발)
    - 새롭게 알게된 내용 최소 2주에 한 번 온라인 공유
    - 월 1회 프로젝트 전체 진행 상황과 앞으로의 방향 오프라인 공유
- 선발 방식
    - (고정) 3~4달 기준 공개 모집(9월 신규 모집 예정)
    - (상시) Project member 중 Bricks member 권한 변경 요청 시 개별 면접 후 진행
- 참고
    - 회비 : 20,000원
    (인프라 사용 + 오프라인 공간 대여 비용)

### **Project member**
- Bricks member가 프로젝트 진행에 필요한 분을 직접 선발한 인원
- 세부 역할
    - 프로젝트 진행에 필요한 일부 권한 부여
    (필요 시 협의 후 추가 권한 부여 예정)
- 주요 활동
    - Bricks memeber와 프로젝트 진행
- 선발 방식
    - 상시 모집
- 참고
    - 회비 : X

### **Reviewer member**
- Bricks member 중 휴식 인원
- 세부 역할
    - 별도의 프로젝트 진행은 하지 않고 공유회(커뮤니티) 활동만 참여하는 인원
- 참고
    - 회비 : 조정 예정
